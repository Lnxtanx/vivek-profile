Rover

How to Built This

1. Materials Required

Gather the following materials to start building your Rover:

  • Cardboard
  • Bottle caps
  • Aluminum-plastic board
  • DC motors (12V, 6 pieces)
  • Jump wires (male-to-male and female-to-female)
  • Arduino Uno
  • L298N motor driver module
  • Bluetooth HC-05 module
  • Lithium battery
  • Double-sided tape
  • Glue stick and glue gun
  • Cardboard cutter
  • PVC pipes

2. Structure and Design

Dimensions:

  • Aluminum plastic board: 24 cm (length) × 17 cm (breadth)
  • PVC pipes: 24 cm (length) × 3 cm (breadth) – 2 pieces
  • Beams:
    • 20 cm (length) × 2 cm (breadth) – 4 pieces
    • 12 cm (length) × 2 cm (breadth) – 2 pieces
  • Tyres:
    • 6 tyres cut from cardboard, each with a radius of 4 cm (1 tyre = 5 layers of cardboard)
    • Hole in each tyre: 3 cm (diameter) × 1.5 cm (radius) × 2 cm (depth)

3. Construction

Building the Structure

Take the aluminum plastic cardboard and attach 2 PVC pipes parallelly along the length of the cardboard using a glue gun, placing them at both corners. Cut the PVC pipes in the middle, leaving a 4 cm section attached to the cardboard for support.

Attach the 20 cm beams at a 120° angle to the cut PVC pipe sections using the glue gun. Repeat on both sides to strengthen the structure. Attach a 12 cm beam to the front of the structure for additional support.

Assemble the tyres:

  • Glue together 5 layers of cardboard to create each tyre.
  • Dig a hole (2 cm depth, 1.5 cm radius) in the center of each tyre and attach a bottle cap to it using a glue gun.
  • Create a hole in the bottle cap to fit the motor nozzle. Use Fevistick for a secure attachment.
  • Fix the motors to the beams using a glue gun, ensuring a ground clearance of 20 cm for all tyres. Position the middle tyres inward-facing.

Adding Electronic Parts

Connect all motors with jump wires. Attach the Arduino Uno to the center of the upper body using double-sided tape and place the battery nearby. Follow these steps for wiring:

  • Group the motors:
    • Connect the positive terminals of 3 motors on one side to form a single positive terminal, and do the same for the negative terminals. Repeat for the other side.
    • Connect the grouped motor terminals to the L298N motor driver.
  • Wire the L298N to the Arduino Uno using male-to-male jump wires:
    • IN1 → Pin 7
    • IN2 → Pin 8
    • IN3 → Pin 9
    • IN4 → Pin 10
  • Connect the HC-05 Bluetooth module to the Arduino Uno using male-to-female wires:
    • TX → TX
    • RX → RX
    • GND → GND
    • VCC → 5V
  • Connect the battery to the L298N and Arduino Uno:
    • GND → 3.3V
    • 5V → VIN
    • +12V → Battery terminal

4. Working Process

Once the construction and electronic setup are complete, follow these steps to operate the rover:

  • Step 1: Power On - Switch on the lithium battery to power the Arduino Uno and L298N motor driver.
  • Step 2: Bluetooth Pairing - Enable Bluetooth on your smartphone or computer. Pair with the HC-05 module.
  • Step 3: Control the Rover - Use a Bluetooth terminal app to send commands:
    • 'F': Move forward
    • 'B': Move backward
    • 'L': Turn left
    • 'R': Turn right
    • 'S': Stop
  • Step 4: Observe - Test the rover's movement on different surfaces and make necessary adjustments if any issues arise.
  • Step 5: Power Off - Turn off the battery after use.

5. Coding

Here is the Arduino code:

/// Complete Arduino Code
const int IN1 = 7;
const int IN2 = 8;
const int IN3 = 9;
const int IN4 = 10;

void setup() {
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  pinMode(IN3, OUTPUT);
  pinMode(IN4, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  if (Serial.available()) {
    char command = Serial.read();
    if (command == 'F') {
      digitalWrite(IN1, HIGH);
      digitalWrite(IN2, LOW);
      digitalWrite(IN3, HIGH);
      digitalWrite(IN4, LOW);
    } else if (command == 'B') {
      digitalWrite(IN1, LOW);
      digitalWrite(IN2, HIGH);
      digitalWrite(IN3, LOW);
      digitalWrite(IN4, HIGH);
    } else if (command == 'L') {
      digitalWrite(IN1, LOW);
      digitalWrite(IN2, HIGH);
      digitalWrite(IN3, HIGH);
      digitalWrite(IN4, LOW);
    } else if (command == 'R') {
      digitalWrite(IN1, HIGH);
      digitalWrite(IN2, LOW);
      digitalWrite(IN3, LOW);
      digitalWrite(IN4, HIGH);
    } else if (command == 'S') {
      digitalWrite(IN1, LOW);
      digitalWrite(IN2, LOW);
      digitalWrite(IN3, LOW);
      digitalWrite(IN4, LOW);
    }
  }
}
 please remeber that this code is used according to pin you connect in arduino-uno
